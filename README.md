# 100 Days of LeetCode

A personal challenge to solve 2 LeetCode problems per day for 100 consecutive days.

## Goal

- 100 days
- 2 problems per day
- 200 problems total

Problems are solved in ascending difficulty order, starting with Easy, then Medium, and finally Hard.

## Repository Structure

Each day contains two solutions.

```text
Day_001/
├── 0001-two-sum.js
└── 0009-palindrome-number.js

Day_002/
├── 0013-roman-to-integer.js
└── 0014-longest-common-prefix.js

...
````

Each file is self-contained and includes:

* Problem information
* Solution implementation
* Example executions

When auxiliary structures are required (e.g. `ListNode`, `TreeNode`), they are defined directly in the solution file rather than shared through helper modules.
